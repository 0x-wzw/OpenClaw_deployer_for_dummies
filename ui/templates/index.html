📋 Next steps

  1. SSH into your server using the command above.
  2. If you didn't enter an API key, edit /opt/openclaw/.env and add your key.
  3. Start the agent: sudo systemctl start openclaw
  4. View logs: sudo journalctl -u openclaw -f
  5. The install log is at: /var/log/openclaw-install.log
